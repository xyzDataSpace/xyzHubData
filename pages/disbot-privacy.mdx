# Privacy Policy for the Discord Bot “disbot”
###### Last Update: 26.11.2025

## 1. Data Controller
The operator of this Discord bot (“we”, “us”) acts as the data controller. 
The bot is self-hosted and privately operated.

## 2. Data Collected
The bot only processes data that is strictly necessary for its functionality.

### The bot may store:
- Discord user IDs (Snowflakes)
- Server (guild) IDs
- Channel or role IDs, if required for specific features
- Bot configuration settings submitted by users or server administrators

### The bot does not store:
- Message content
- Attachments
- IP addresses
- Personal identification data
- Behavioral or analytics data
- Any metadata beyond what is required for core functionality

## 3. Storage Location
All data is stored locally in a PostgreSQL database.
There are no connections to external services, cloud storage, tracking systems, or third-party APIs.

No data is transmitted to third parties under any circumstances.

## 4. Logging
The bot does not create or store personal logs.
This includes:
- No message logs
- No command logs
- No user action logs

Only minimal technical logs may exist for debugging, but they contain no personal data.

## 5. Data Deletion
Users may request the deletion of their data at any time.
Server administrators may also request the removal of all data associated with their server.

Data is automatically deleted when:
- The bot is removed from a server
- The stored data is no longer required for bot functionality
